## [2026.06.09-ai-agent115] — 2026-06-09

### 新增
- 通用采集中心首次发布
- 一键采集工作台（普通 / 深度 / 完整三种模式）
- AI 抓取工作台（18 个厂商预设、字段建议、自然语言任务规划）
- 批量采集（真实浏览器、子页面爬取、任务队列、失败恢复）
- 模板库 + 模板市场 + 场景模板预设
- 历史与监控（采集历史、任务档案、变更提醒、变更报告、计划采集）
- 采集结果完整度评分和诊断建议
- 重抓低完整度、重抓分页、重抓子链接、AI 修字段
- 抽样验证和实测对比
- 定时监控（计划采集）
- 文件转表格（PDF / 图片 / TXT / CSV）
- 邮箱和电话提取
- 图片下载
- 系统托盘变更通知
- 旧版闲鱼监测兼容入口（`--xianyu`）

### 架构
- UI 层通过 `@register` 装饰器注入方法，支持 40+ 个独立 UI 子模块
- 核心业务逻辑集中在 `universal_core.py`
- SQLite 数据库、运行档案、变更报告查询在 `core_database.py`
- AI Key 使用 Windows DPAPI 加密存储（`core_ai_storage.py`）
- Firecrawl 远程/自托管爬取适配（`core_firecrawl.py` / `core_firecrawl_flow.py`）

### 已知问题
- 记忆宫殿功能为桩实现（`ui_history_change.py`），后续版本完善
- `universal_ui.py` 和 `universal_core.py` 文件偏大，后续继续拆分
