🧠 智能找资料

使用内置演示数据测试发现流程。

输入主题后，系统自动生成多组搜索查询，从公开网络资料源中发现候选资料。搜索 API 未配置时使用演示数据。

支持 CSV（含 url 列）、JSON 数组、TXT（每行一个 URL）、Markdown 文件。

仅读取公开 sitemap。不绕过登录或访问控制。支持 sitemap index 递归。

支持 RSS 2.0 和 Atom 格式。仅读取公开 feed。

输入域名后，系统会自动发现 robots.txt / sitemap / RSS / 常见资料入口，生成候选 URL。
