第 2 步：合规确认

i 系统只允许在公开、授权、无需登录、无需验证码、无需 Cookie/Token 的页面上进行测试采集。

合规清单

✓
页面公开可访问
目标页面无需登录即可在浏览器中打开查看。
✓
用户有权访问
仅采集你拥有合法访问权限的页面。
✓
不需要登录
不通过任何账号体系绕过访问限制。
✓
不需要验证码
不破解、不绕过任何人机验证。
✓
不需要 Cookie / Token
不使用认证凭据模拟登录态。
✓
不绕过访问控制
遵守目标站点的 robots.txt 和访问规则。
✓
不规避反爬机制
不使用代理池、伪造 UA、JS 逆向等手段。
✓
不高频请求
每次请求间隔 ≥2 秒，不干扰目标站点正常运行。
