# ACS 资料采集助手 — 桌面使用说明

## 版本

ACS v1.1.0-discovery 安全测试模式

---

## 如何启动

### 方式一：双击启动（推荐）

```
双击 start_acs_desktop.bat
```

### 方式二：命令行启动

```bash
python start_acs_desktop.py
```

启动参数：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `--port` | 5020 | 本地服务端口 |
| `--host` | 127.0.0.1 | 绑定地址（仅本地） |
| `--no-browser` | false | 不自动打开浏览器 |
| `--timeout` | 30 | 等待服务就绪超时秒数 |

---

## 启动后会发生什么

1. 检查 Python + 依赖 (Flask)
2. 启动本地服务 `127.0.0.1:5020`（仅本地，不暴露网络）
3. 等待服务健康检查通过
4. 自动打开浏览器 → ACS 资料采集助手界面
5. 首页显示"本地服务已连接"，可以开始使用

---

## 如何使用

### 主流程：智能找资料

```
1. 点击"智能找资料"
2. 选择资料发现方式（Mock演示 / 导入文件 / Sitemap / RSS）
3. 输入主题和关键词
4. 点击"开始找资料"
5. 候选来源确认 — 勾选需要的条目（blocked 项不可选）
6. 点击"加入采集任务"
7. 点击"开始测试采集"
8. 查看进度 → 结果表格
9. 导出 JSON / CSV / Markdown
```

### 其他功能

| 功能 | 入口 |
| --- | --- |
| 新建采集任务 | 导航 → 新建采集 |
| 导入 URL 文件 | 导航 → 资料来源 |
| 字段规则 | 导航 → 字段规则 |
| 导出报告 | 采集结果页 → 导出 JSON/CSV/Markdown |
| 高级设置 | 导航 → 高级设置 |

---

## 如何关闭

1. 关闭浏览器页面
2. 在启动窗口按 `Ctrl+C`
3. 服务自动停止

---

## 常见问题

### 端口被占用

```
错误: 端口 5020 已被占用
解决: 关闭其他占用 5020 端口的程序，或使用 --port 5030 指定其他端口
```

### 服务未连接

```
页面显示"本地服务未连接"
解决: 运行 start_acs_desktop.bat 启动服务
```

### 打不开页面

```
检查: start_acs_desktop.bat 窗口是否还在运行
检查: 是否有杀毒软件拦截
```

### 路径含中文

```
当前项目已支持中文路径。如仍有问题，请将项目放到不含中文的路径下。
```

### 依赖缺失

```
提示: 缺少依赖 flask
解决: pip install flask
```
